# Ratelimit
This page documents the ratelimit configuration of Zipline.

## `RATELIMIT_USER`
The amount of seconds a user has to wait before upload another image.
```bash
RATELIMIT_USER=5
```

## `RATELIMIT_ADMIN`
The amount of seconds a admin has to wait before upload another image.
```bash
RATELIMIT_ADMIN=1
```
