title: Core Config
---

## `core.secure`
Whether or not to use https or not

**Type**: `boolean`
**Example:**:
```js
true
```

## `core.secret`
Whether or not to use https or not

**Type**: `string`
**Example:**:
```js
'1234567890qwertyuiopasdfghjklzxcvbnm'
```

## `core.host`
What host Zipline should run on.
:::info
If using Docker, set the value to `'0.0.0.0'`
:::

**Type**: `string`
**Example:**:
```js
'0.0.0.0'
```

## `core.port`
What port Zipline should run on

**Type**: `number`
**Example:**:
```js
3000
```
