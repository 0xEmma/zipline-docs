# Ratelimit

## `ratelimit.user`
Ratelimit in seconds for normal user accounts

**Type**: `number`

**Env Property**: `RATELIMIT_USER`

**Example**:
```js
5
```

## `ratelimit.admin`
Ratelimit in seconds for administrator accounts

**Type**: `number`

**Env Property**: `RATELIMIT_ADMIN`

**Example**:
```js
5
```
