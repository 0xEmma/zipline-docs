# Zipline
Zipline is a file sharing, URL sharing, lightweight and easy to use! Setup Zipline with only one command, the easiest it has been.

## Configurable  
Zipline allows you to configure every part of it.    

## Speed 
Zipline provides a speedy web app & a speedy api to upload your images.    

## Built with Next.Js
Zipline is built on Next.js & React, to provide a fast web interface.
